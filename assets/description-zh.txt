R2Shot — 一键截图上传至 Cloudflare R2 ⚡

只需一次点击，即可截取浏览器标签页，自动上传至您的 Cloudflare R2 存储桶，CDN 链接直接复制到剪贴板。适合开发者、设计师以及需要快速可靠截图托管的所有人。

✨ 功能特性

📸 一键截图 — 从浏览器工具栏截取任意标签页的可见区域
☁️ Cloudflare R2 上传 — 通过 S3 兼容 API 自动上传，按日期文件夹/GUID 命名
🔗 CDN 链接 — 使用自定义域名生成公开链接，一键复制到剪贴板
🧠 智能端点解析 — 粘贴完整的 S3 API URL，自动提取端点和存储桶名称
🔍 连接测试 — 保存前验证 R2 凭证是否有效
🎨 主题支持 — 浅色、深色或跟随系统
⚙️ 可调质量 — JPG 压缩质量 1-100 可调
🌍 多语言 — 支持 10 种语言

🛡️ 隐私说明

R2Shot 完全在本地运行，不收集、追踪或与第三方共享任何数据。截图仅上传至您自己的 Cloudflare R2 存储桶。所有配置存储在浏览器本地。

隐私政策：https://github.com/nocoo/r2shot/blob/main/PRIVACY.md
