R2Shot — Captura de tela com um clique para Cloudflare R2

Capture qualquer aba do navegador com um único clique, faça upload automático para seu bucket Cloudflare R2 e tenha uma URL CDN copiada para sua área de transferência. Perfeito para desenvolvedores, designers e qualquer pessoa que precise de hospedagem de capturas de tela rápida e confiável.

Recursos

- Captura com um clique — capture a área visível de qualquer aba pela barra de ferramentas
- Upload Cloudflare R2 — upload automático via API compatível com S3, nomeação por pasta de data/GUID
- URL CDN — gera URL pública com seu domínio personalizado, cópia com um clique
- Análise inteligente do endpoint — cole uma URL API S3 completa para extrair automaticamente o endpoint e nome do bucket
- Teste de conexão — verifique suas credenciais R2 antes de salvar
- Temas — claro, escuro ou sistema
- Qualidade configurável — compressão JPG de 1 a 100
- Multilíngue — 10 idiomas suportados

Privacidade

R2Shot funciona inteiramente no seu dispositivo. Nenhum dado é coletado, rastreado ou compartilhado.
