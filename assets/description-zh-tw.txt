R2Shot — 一鍵截圖上傳至 Cloudflare R2

只需一次點擊，即可截取瀏覽器標籤頁，自動上傳至您的 Cloudflare R2 儲存桶，CDN 連結直接複製到剪貼簿。適合開發者、設計師以及需要快速可靠截圖託管的所有人。

功能特性

- 一鍵截圖 — 從瀏覽器工具列截取任意標籤頁的可見區域
- Cloudflare R2 上傳 — 透過 S3 相容 API 自動上傳，按日期資料夾/GUID 命名
- CDN 連結 — 使用自訂網域產生公開連結，一鍵複製到剪貼簿
- 智慧端點解析 — 貼上完整的 S3 API URL，自動擷取端點和儲存桶名稱
- 連線測試 — 儲存前驗證 R2 憑證是否有效
- 主題支援 — 淺色、深色或跟隨系統
- 可調品質 — JPG 壓縮品質 1-100 可調
- 多語言 — 支援 10 種語言

隱私說明

R2Shot 完全在本機運行，不收集、追蹤或與第三方共享任何資料。截圖僅上傳至您自己的 Cloudflare R2 儲存桶。
