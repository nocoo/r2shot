R2Shot — 원클릭 스크린샷을 Cloudflare R2에 업로드

브라우저 탭을 한 번의 클릭으로 캡처하고 Cloudflare R2 버킷에 자동 업로드합니다. CDN URL이 클립보드에 복사됩니다. 개발자, 디자이너 및 빠르고 안정적인 스크린샷 호스팅이 필요한 모든 분께 적합합니다.

기능

- 원클릭 캡처 — 브라우저 툴바에서 탭의 보이는 영역 스크린샷
- Cloudflare R2 업로드 — S3 호환 API로 자동 업로드, 날짜 폴더/GUID 명명
- CDN URL — 사용자 지정 도메인으로 공개 URL 생성, 원클릭 클립보드 복사
- 스마트 엔드포인트 파싱 — S3 API URL을 붙여넣으면 엔드포인트와 버킷 이름 자동 추출
- 연결 테스트 — 저장 전 R2 자격 증명 확인
- 테마 — 라이트, 다크, 시스템 설정 지원
- 품질 조절 — JPG 압축 품질 1~100 조절 가능
- 다국어 — 10개 언어 지원

개인정보

R2Shot은 모든 작업이 로컬에서 이루어집니다. 데이터 수집, 추적, 제3자 공유가 없습니다.
