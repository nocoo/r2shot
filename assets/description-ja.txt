R2Shot — ワンクリックで Cloudflare R2 にスクリーンショット ⚡

ブラウザのタブをワンクリックでキャプチャし、Cloudflare R2 バケットに自動アップロード。CDN URL がクリップボードにコピーされます。開発者、デザイナー、高速で信頼性の高いスクリーンショットホスティングが必要な方に最適です。

✨ 機能

📸 ワンクリックキャプチャ — ブラウザツールバーから任意のタブの表示領域をスクリーンショット
☁️ Cloudflare R2 アップロード — S3 互換 API で自動アップロード、日付フォルダ/GUID で整理
🔗 CDN URL — カスタムドメインで公開 URL を生成、ワンクリックでクリップボードにコピー
🧠 スマートエンドポイント解析 — S3 API URL を貼り付けるだけでエンドポイントとバケット名を自動抽出
🔍 接続テスト — 保存前に R2 認証情報を検証
🎨 テーマ — ライト、ダーク、システム設定に対応
⚙️ 品質調整 — JPG 圧縮品質を 1〜100 で調整可能
🌍 多言語対応 — 10 言語をサポート

🛡️ プライバシー

R2Shot はすべてローカルで動作します。データの収集、追跡、第三者との共有は一切ありません。スクリーンショットはお客様ご自身の Cloudflare R2 バケットにのみアップロードされます。

プライバシーポリシー：https://github.com/nocoo/r2shot/blob/main/PRIVACY.md
