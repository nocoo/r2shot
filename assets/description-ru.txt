R2Shot — Снимок экрана в один клик в Cloudflare R2

Захватите любую вкладку браузера одним щелчком, автоматически загрузите в ваш бакет Cloudflare R2 и получите CDN-ссылку, скопированную в буфер обмена. Идеально подходит для разработчиков, дизайнеров и всех, кому нужен быстрый и надёжный хостинг скриншотов.

Возможности

- Захват одним кликом — скриншот видимой области любой вкладки с панели инструментов
- Загрузка в Cloudflare R2 — автоматическая загрузка через S3-совместимый API, именование по дате/GUID
- CDN URL — генерирует публичную ссылку с вашим доменом, копирование одним кликом
- Умный разбор эндпоинта — вставьте полный URL S3 API для автоматического извлечения эндпоинта и имени бакета
- Тест соединения — проверьте учётные данные R2 перед сохранением
- Темы — светлая, тёмная или системная
- Настраиваемое качество — сжатие JPG от 1 до 100
- Мультиязычность — поддержка 10 языков

Конфиденциальность

R2Shot работает полностью на вашем устройстве. Никакие данные не собираются, не отслеживаются и не передаются.
